# Analyse DTW des Passagers Aériens (1949-1960)

## Description
Analyse complète de la série temporelle des passagers aériens mensuels utilisant Dynamic Time Warping (DTW).

## Objectifs
- Analyser la tendance et saisonnalité des données
- Appliquer DTW pour comparer les années
- Visualiser les alignements temporels

## Structure
- `Analyse_DTW_Passagers_Aeriens.ipynb` : Notebook Jupyter avec l'analyse complète
- `air_passengers.csv` : Données originales
- `images_dtw/` : Visualisations générées
- `TP_DTW_INF4117.pdf` : Présentation en LaTeX sur DTW
- `24F2456_ESSUTHI_MABNGUE_ANGE_ARMEL.md` : Cahier de suivi individuel

## Utilisation
Ouvrir le notebook Jupyter et exécuter les cellules dans l'ordre.

## Résultats Clés
- Années les plus similaires : 1956-1957
- Évolution de la stabilité saisonnière post-1955
- Matrice complète des distances DTW
